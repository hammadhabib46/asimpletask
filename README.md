# TaskFlow - Simple Task Management

A clean, minimal task management app for teams built with Next.js, Convex, Shadcn UI, and Clerk.

## Features

- **Admin Dashboard**: Create teams, projects, and tasks. Assign tasks to team members.
- **Employee Dashboard**: View assigned tasks with search, filtering, and mark as done.
- **Real-time Sync**: Powered by Convex for instant updates.
- **Clean UI**: Minimal, modern design using Shadcn UI components.

## Setup

### 1. Install Dependencies

```bash
npm install
```

### 2. Configure Clerk

1. Create an account at [clerk.com](https://clerk.com)
2. Create a new application
3. Copy your API keys

### 3. Configure Convex

```bash
npx convex dev
```

This will:
- Prompt you to log in to Convex
- Create a new project
- Deploy your backend functions
- Generate type definitions

### 4. Set Environment Variables

Create a `.env.local` file:

```env
# Clerk
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_...
CLERK_SECRET_KEY=sk_test_...
NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up

# Convex (auto-generated by npx convex dev)
NEXT_PUBLIC_CONVEX_URL=https://your-project.convex.cloud
```

### 5. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Usage

### As an Admin

1. Sign up and select "Admin"
2. Enter your team name
3. Add team members by their email
4. Create projects
5. Add tasks to projects and assign them

### As an Employee

1. Sign up (using email added by admin) and select "Employee"
2. View your assigned tasks
3. Use search and filters to find tasks
4. Mark tasks as done

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Backend**: Convex
- **Auth**: Clerk
- **UI**: Shadcn UI + Tailwind CSS
- **Language**: TypeScript
